## 2. 易错点

- **布局**：`App.vue` 的 `<router-view />` 勿包 `d-flex`；对话壳用 `chat_pane` Grid（`HomeChat` / `useChatSession`）。
- **数据**：chunk **`meta.links` 为权威**；`index` head/tail 可 repair（`DOC/08`）。
- **Memory**：chunk JSON 权威；Lance 在 `{userId}/memory/conversations/{id}/`。**同一 `DATA_DIR` 仅一个 server 进程**（prod+dev 双开会损坏 Lance）。
- **Hybrid FTS**：全局 `user-preferences.json` → `hybridFts`；`zh-jieba` 词典在 `{userId}/hybrid-fts/zh-jieba/{variant}/`；Lance 经 **`LANCE_LANGUAGE_MODEL_HOME`** 定位（`withLanceLanguageModelHome`，勿假设可直传 dict 路径）。详见 `DOC/03` §14.4.3。
- **API Key**：浏览器 GET 无明文；chat/models 服务端 resolve（`DOC/25` §15）。
- **Historian**：插件 id `plot-summary`；自动块指针 `nextBlockStart` / `lastSummarizedEnd`；条目标题 `[MEMO-n]-TITLE-[from-to]`（`DOC/12` §4.2）。
- **宿主零特化（强制）**：**`DOC/41`** + **`DOC/42`**（审计/checklist）+ `.cursor/rules/plugin-host-generic.mdc` — 宿主禁止 bundled id 特化；`npm run check:host-no-plugin-ids`；迁移 **`DOC/04` §宿主去特化**。
