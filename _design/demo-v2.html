v2 新增（基于反馈）

  • 头像（avatar）：圆形 30×30，EB Garamond 衬线斜体字母 + 黄铜环（user）/ 赤土橙环（assistant），客栈名牌感。未来可替换为真实图片，CSS 接口已留好。
  • 用户名行插件槽（plugin slots）：每条消息的角色名行尾部预留 3-4 个 .plugin-slot 占位按钮——虚线圆环 = 空槽位、实线 = 已激活插件（如收藏 / 思维链 / RAG 命中）。容器带 data-plugin-slot="user-turn|assistant-turn"，对接 Vue 时 plugin 系统可直接 query 到位置。
  • 气泡下 toolbar 全 icon-only：28×28 方按钮，去掉文字，hover 弹出 tooltip（暖墨黑底 + 上箭头 + 1px hairline），危险动作（删除）hover 变酒红。
  • Tooltip 设计：纯 CSS [data-tt] 属性驱动，落到 Vue 时换成 v-tooltip 即可，DOM 结构和样式都不动。Demo 上为了截图能看到，第二轮 assistant 的「再生」按钮做了 .show-tt force 显示。

保留的 v1 DNA

  • 骨架（Linear）：1px hairline 极细分隔、暖墨黑 #0E0B08 底、信息密度高、键盘 hint chip、tabular 数字。
  • 灵魂（Tavern）：赤土橙 #D9602E + 黄铜 #B89770、衬线 EB Garamond display、章回 ❦ 分隔、卡片左侧"门帘竖线"。
  • 字型组合：UI / 数据 Inter；Display / 章回 / brand EB Garamond；标签 / 数字 Geist Mono。
  • 反 AI slop：4 色系（暖墨黑、羊皮米、赤土橙、黄铜）+ 拒绝紫渐变 / emoji icon / 圆角左 border accent。
