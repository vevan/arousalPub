v3 新增（基于反馈）

  • 头像放大到 4rem (64×64)：从"标签级"升到"视觉主角"，2px 黄铜/赤土橙边环、内嵌径向高光、外阴影增强立体。EB Garamond italic 大写字母占位，未来 <img> 直接换实图即可。
  • Grid 排版：每个 .turn 用 CSS Grid 排——头像独立成列贴顶（align-self: start），右/左侧依次堆叠 role / reasoning / bubble / toolbar / swipe。User 用反向列序。气泡新增了 ::before 小三角指向头像。
  • 可折叠思维链：用原生 <details> + 自定义样式——Mono 大写小字 summary（橙色，与 bubble 区分）、衬线斜体 body（"内心独白"感）、左侧 2px 暖色细线、步骤分 01 / 02 / 03 编号。展开/折叠靠 [open] 选择器切换 caret 箭头方向，文字 hint 同步切换"展开/收起"，按 R 可快捷切换（暗示快捷键）。Demo 里第 1 回展开、第 2 回折叠做对比演示。

v3 排版核心 · 给落 Vue 的同学

  • Grid 模板：assistant 用 4rem / 1fr，user 用 1fr / 4rem。grid-template-areas 让所有 turn 内的子组件位置一致，未来加新组件（如 token 计数、cost 标签）只要给一个 area 名字。
  • 思维链组件接口：<details> 的 open 属性绑 Vue ref；summary 内 .reasoning-chain__caret（SVG）+ .reasoning-chain__title（带 meta 步数/时间）+ .reasoning-chain__hint（带 kbd）三部分都是稳定 anchor 点。body 里的 .reasoning-step 可循环渲染。
  • 气泡小三角：纯 CSS ::before 实现，没有 SVG，配色随气泡变量，便于换 light theme。

保留的 v1 / v2 DNA

  • 骨架（Linear）：1px hairline 极细分隔、暖墨黑 #0E0B08 底、键盘 hint chip、tabular 数字。
  • 灵魂（Tavern）：赤土橙 #D9602E + 黄铜 #B89770、衬线 display、章回 ❦ 分隔、卡片左侧"门帘竖线"。
  • 插件槽（v2 引入）：角色名行尾部 .plugin-slots[data-plugin-slot]，虚线圆 = 空、实线 = 已激活。
  • 气泡下 toolbar icon-only + tooltip（v2）：纯 CSS [data-tt] 驱动，落 Vue 时换成 v-tooltip。
